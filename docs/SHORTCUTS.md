# Keyboard Shortcuts & Mouse Actions

All controls are also available in the sidebar. Press `H` in the viewer to show this help.

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Q` | Quit |
| `S` | Save screenshot (PNG + 32-bit float °C radiometric TIFF) |
| `P` | Cycle color palette |
| `F` | Flip image horizontally |
| `R` | Rotate image 90° clockwise |
| `Ctrl+R` | Toggle video recording |
| `C` | Trigger shutter calibration |
| `N` | Trigger NUC (non-uniformity correction) |
| `G` | Toggle temperature graph panel |
| `M` | Toggle mosaic view (all 6 palettes) |
| `3` | Toggle 3D surface plot |
| `L` | Lock/unlock color range |
| `D` | Delete nearest probe point |
| `+` / `-` | Adjust display brightness |
| `A` / `Shift+A` | Increase / decrease contrast |
| `E` / `Shift+E` | Increase / decrease emissivity |
| `Esc` | Clear selection and probe points |
| `H` | Show keyboard shortcuts help |

## Mouse Actions

| Action | Effect |
|--------|--------|
| Hover | Show temperature at cursor |
| Left-click | Pin a probe point (up to 10) |
| Double-click | Remove nearest probe point |
| Left-drag | Select rectangular ROI |
| Right-click | Clear ROI selection |

### 3D Surface View

| Action | Effect |
|--------|--------|
| Left-drag | Rotate view |
| Ctrl+Left-drag | Pan view |
| Middle-drag | Pan view |
| Scroll wheel | Zoom in/out |
